1) Copy client/env.sample to client/.env (no changes need now)

2) Build Client (cd client; npm install; npm run build)

3) Start ngrok (port 3000), copy your url

4) Create Twilio API Key

5) create Twilio TwiML APP, point the url to POST to your ngrok url + "/twiml" (https://your-subdomain.ngrok.app/twiml)

6) copy server/start-local-server.sh.copy to server/start-local-server.sh and set YOUR env variables in server/start-local-server.sh

7) Start server (cd server, npm install, ./start-local-server.sh )

8) Web browser http://localhost:3000, watch things happen in the terminal window and the data directory....
