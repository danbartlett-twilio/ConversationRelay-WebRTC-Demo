1) Build Client (cd client; npm install; npm run build)

2) Start ngrok (port 3000), copy your url

3) Create Twilio API Key

4) create Twilio TwiML APP, point the url to POST to your ngrok url + "/twiml" (https://your-subdomain.ngrok.app/twiml)

5) copy server/start-local-server.sh.copy to server/start-local-server.sh and set YOUR env variables in server/start-local-server.sh

6) Start server (cd server, npm install, ./start-local-server.sh )

7) Web browser http://localhost:3000, watch things happen in the terminal window and the data directory....
